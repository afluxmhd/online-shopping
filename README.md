# Installation

1. Install XAMPP or WAMPP.

2. Open XAMPP Control panel and start **Apache** and **MySQL**.

3. follow Git Bash commands:
   
    i. `cd C:\\xampp\htdocs\`
    
    ii. `git clone https://github.com/afluxmhd/online-shopping-system-with-advanced-admin-page.git`

4. Extract files in `C:\\xampp\htdocs\`.

5. Open the link [localhost/phpmyadmin](http://localhost/phpmyadmin).

6. Click on **New** in the side navbar.

7. Give a database name as `onlineshop` and hit the **Create** button.

8. After creating the database name, click on **Import**.

9. Browse the file in the directory `online-shopping-system-with-advanced-admin-page/database/onlineshop.sql`.

10. After importing successfully.

11. Open any browser and type `http://localhost/online-shopping-system-with-advanced-admin-page-master`.

12. First register and then login.

13. Admin login details: Email=admin@gmail.com or Username=admin and Password=123456789.
